# calculator 

installing dependencies - `npm i`

Math calculator with arithmetic operations

test script - `npm run test`

start script - `npm run calc`

Then you can calc math operations

Examples:

> 10 + 10
Result: 20

> 10 + 10 * 20 - 30
Result: 180

> 19 + -10
Result: 9

> 3 ** 
Result: 9

> 5 !
Result: 120
